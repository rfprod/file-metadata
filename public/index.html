  • User can pass a URL as a parameter like and will receive a shortened URL in the JSON response.
  • Create shortened URL examples:
      if URL actually exists
    • https://url-shortener-rfprod.c9users.io/new/https://iana.org

    • { "original_url": "http://freecodecamp.com/news", "short_url": "https://shurli.herokuapp.com/1" }
