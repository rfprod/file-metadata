URL Shortener Microservice
  • User can pass a URL as a parameter like and will receive a shortened URL in the JSON response.
  • Create shortened URL examples:
      if URL actually exists
    • https://url-shortener-rfprod.c9users.io/new/https://iana.org
      if URL does not actually exist
    • https://url-shortener-rfprod.c9users.io/new/invalidURL?allow=true
  • Example JSON output:
    • { "original_url": "https://iana.org", "short_url": "https://url-shortener-rfprod.c9users.io/1" }
  • URL shortening results retrieval:
    • https://url-shortener-rfprod.c9users.io/1
    • https://url-shortener-rfprod.c9users.io/2
    • https://url-shortener-rfprod.c9users.io/3
  • Debug utility:
    • Show all shortened URLs
